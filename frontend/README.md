# Hazy Factory Frontend

This is a modern, responsive landing page for the Hazy Content Factory.

## Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- **Next.js 14** (App Router)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Vanilla CSS Modules** (Styling)

## Notes
- The "Hazy Modern" theme is defined in `src/app/globals.css`.
- The main landing page is in `src/app/page.tsx`.
